## JALANKAN APLIKASI

Untuk menjalankan aplikasi :
- Buat Database kaspulsa
- php artisan migrate
- php artisan storage:link
- php artisan db:seed --class=RoleSeeder
- php artisan db:seed --class=UserSeeder
- php artisan serve
